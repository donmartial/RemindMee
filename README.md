# RemindMee

RemindMee is a macOS menu bar productivity app that helps you stay healthy and focused during work with break reminders and a lightweight daily task list.

## What RemindMee Helps With

- **Stay active while working**
  - Stand-up reminders at your chosen interval.
  - Optional smart break suggestions with rotating activity ideas.
- **Manage today's tasks quickly**
  - Add, edit, complete, and delete tasks from the menu bar.
  - Set priority and due date for each task.
  - Filter done tasks and use bulk actions.
- **Reduce interruptions when busy**
  - Pause reminders during Focus mode (heuristic).
  - Pause reminders during current or upcoming calendar events.
- **Personalize your experience**
  - Choose reminder interval and notification behavior.
  - Toggle smart break suggestions.
  - Customize the menu bar icon.

## Key Highlights

- Built for fast, low-friction interaction from the macOS menu bar.
- Designed for daily use with simple controls and minimal setup.
- Keeps data locally on your machine.

## Permissions

- **Notifications**: to deliver stand-up reminders.
- **Calendar access**: to detect active/upcoming meetings and suppress reminders when appropriate.

If permissions are denied, RemindMee still works with reduced related functionality.

## Keyboard Shortcuts

- `Cmd+N`: open add-task UI from menu bar panel
- `Cmd+,`: open settings window
- `Cmd+Q`: quit app
- `Cmd+Shift+A`: mark all today's tasks as done (bulk action)
